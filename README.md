## Sobre o Projeto

Sistema web desenvolvido para a comercialização de bois, utilizando PHP, HTML e CSS, seguindo boas práticas de desenvolvimento de websites comerciais.

A aplicação permite o gerenciamento e a visualização dos animais disponíveis para venda, simulando um ambiente real de e-commerce. O projeto teve como objetivo aplicar os conceitos aprendidos em sala de aula, fortalecendo a experiência prática no desenvolvimento de soluções comerciais.
